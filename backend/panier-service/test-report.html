Rapport de tests unitaires

Après exécution des tests, ouvrez target/site/surefire-report.html pour voir le rapport détaillé.

Voir le rapport généré

Étapes pour générer et visualiser le rapport

  1. Ouvrez un terminal à la racine du projet.
  2. Lancez les tests avec : mvn test
  3. Générez le rapport HTML avec : mvn surefire-report:report
  4. Ouvrez le fichier target/site/surefire-report.html dans votre navigateur.
